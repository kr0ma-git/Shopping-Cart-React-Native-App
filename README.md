# Shopping Cart Mobile App

#### This project uses Expo and Expo Go (Mobile App) for development, testing, and debugging.

- Install Expo Go on either your Android and iPhone device.
- On the main project directory, type the following in a terminal:

```bash
    npx expo start
```

- This should generate a QR code.
- In the Expo Go App, scan the QR code to begin the live development session.
- All changes made to the code updates on the Expo Go App (neat!).

#### IF the above expo command does not start a session or does not work in general:

1. Delete pre-existing package-lock.json
2. Delete pre-existing node_modules

#### Then run in the root directory's terminal:

```bash
    npm install
```

##### This should reinstall all necessary node_modules.
